# 문자

character 를 뜻하는 char 자료형은 문자를 표현할 수 있는 자료형이다.

```.c
char c = 'a';
printf("%c", c);
```

이와 같이 문자를 표현할 때에는 ' ' 로 묶어주고 %c 로 출력한다.

```.c
char c = 'a';
printf("%d", c);
```

정수형태로 출력하면 97이 나온다. 이는 오류가 아니다.

컴퓨터는 문자를 인식하지 못한다. 하지만 사람은 문자를 표현해야하기 때문에 숫자에 각 문자를 대입시켰다.
